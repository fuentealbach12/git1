### Actividad: Instalación y comandos básicos de Git

#### Comandos Más Importantes

*   **`git --version`**: Muestra qué versión está instalada.
    *   *Uso:* `git --version`
*   **`git init`**: Inicializa el repositorio.
    *   *Uso:* `git init`
*   **`git add`**: Agrega los cambios.
    *   *Uso:* `git add .` (agrega todos los archivos de la carpeta donde se ejecuta)
*   **`git commit`**: Registra los cambios con un mensaje.
    *   *Uso:* `git commit -m "Primera  subida"`
*   **`git status`**: Estado del repositorio.
    *   *Uso:* `git status`
